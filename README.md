# [MY WORD@@@@@@@@배포링크첨부안함](배포링크)

## 프로젝트 소개
**단어장과 퀴즈를 제공하여 단어 암기에 유용한 웹사이트를 구현하였습니다.**

## 구현 기능
- 단어 저장 · 읽기 · 수정 · 삭제
- 암기 확인용 단어와 뜻 숨김 기능
- 퀴즈를 풀기 전 단어 맞추기, 뜻 맞추기, 문제 수 설정
- 단어 퀴즈


## 기술
#### 프론트엔드
- React  
- CSS  
- LocalStorage  

#### 배포
- netlify
